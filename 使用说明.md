## 1. 登录与账号管理

### 1.1 登录
- 访问应用地址后，会跳转到登录页面
- 输入用户名和密码，点击"登录"按钮
- 成功登录后进入主界面

### 1.2 账号类型
| 账号类型 | 说明 |
|---------|------|
| 超级管理员 | 可查看总访问量、管理用户、无机器码限制 |
| 普通用户 | 只能使用分析功能，有机器码7天有效期限制 |

### 1.3 用户管理（仅管理员）
- 点击主界面右上角的"👥 用户管理"进入
- **功能**：
  - 新增用户：可设置用户名、密码、有效期（可选）、备注
  - 查看用户列表（包含密码、访问次数、状态）
  - 切换用户状态（正常/禁用）
  - 删除用户

#### 新增用户说明
| 字段 | 必填 | 说明 |
|------|------|------|
| 用户名 | ✅ | 至少3个字符，不能与现有用户重复 |
| 密码 | ✅ | 至少8个字符 |
| 有效期 | ❌ | 可选，不填则永久有效 |
| 备注 | ❌ | 可选，用于记录用户信息 |

#### 用户列表说明
| 字段 | 说明 |
|------|------|
| 用户名 | 用户登录账号 |
| 密码 | 明文显示（请妥善保管） |
| 角色 | 管理员或普通用户 |
| 有效期 | 用户账号的有效期，永久或指定日期 |
| 状态 | 正常/禁用，可点击切换 |
| 访问次数 | 该用户的登录次数统计 |
| 备注 | 创建时填写的备注信息 |
| 创建时间 | 账号创建时间 |
| 操作 | 删除用户（仅普通用户可删除） |

## 3. 参数说明

### 3.1 基本参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| **股票代码** | 600036 | - | 6位数字（A股），例如：600036（招商银行）、000001（平安银行）、600519（贵州茅台） |
| **复权方式** | qfq | qfq/hfq | qfq=前复权，hfq=后复权 |
| **开始时间** | 一年前 | - | 数据开始日期，默认今日往前推365天 |
| **结束时间** | 今日 | - | 数据结束日期，默认为当日 |

### 3.2 分析参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| **极值窗口** | 7 | 3-20 | 识别局部高点/低点的观察范围<br>数值越大，识别的极值点越少<br>推荐：短期3-5，中期7-10，长期12-20 |
| **最大聚类数** | 8 | 3-15 | K-Means聚类的最大聚类数<br>数值越大，价格层级越细<br>推荐：波动小的股票5-8，波动大的股票10-15 |
| **阈值比例** | 5 | 1-10 | 支撑压力位的有效距离范围（%）<br>超出此范围的支撑压力位不显示 |
| **成交量分箱** | 25 | 15-50 | 成交量剖面的价格区间数量<br>数值越大，分辨率越高<br>推荐：数据少（<100天）15-20，数据多（>200天）30-50 |

## 4. 技术指标解读

### 4.1 分析流程

1. 输入股票代码和参数
2. 点击"开始分析"按钮
3. 等待几秒钟，系统自动获取数据并分析
4. 在"📊 分析结果"选项卡查看文字说明
5. 在"📈 箱体图表"选项卡查看可视化图表

### 4.2 分析结果 - 文字说明

#### 当前价格
- 显示股票最新收盘价
- 单位：人民币（元）

#### 主要支撑位
- **价格**：主要支撑位价格
- **置信度**：0.0-1.0，数值越高越可靠
- **说明**：价格下跌时可能反弹的位置，重要的买入参考点

#### 主要压力位
- **价格**：主要压力位价格
- **置信度**：0.0-1.0，数值越高越可靠
- **说明**：价格上涨时可能回落的位置，重要的卖出参考点

#### 成交量剖面 (Volume Profile)
| 指标 | 说明 |
|------|------|
| **POC (Point of Control)** | 成交量最大的价格位，是最重要的支撑压力位<br>价格接近POC时往往会有较强的支撑或阻力 |
| **VA (Value Area)** | 成交量占70%的价格区间<br>VA Low=价值区域下限<br>VA High=价值区域上限<br>价格在VA内运行是正常的，突破VA可能预示趋势变化 |

#### 次要支撑/压力位
- 除主要支撑压力位外的其他价格层级
- 按价格从低到高排序
- 可以作为辅助参考

## 5. 图表分析

点击"📈 箱体图表"选项卡，查看完整的可视化图表。

### 5.1 主图（上半部分）

#### 颜色说明
| 颜色/样式 | 含义 |
|----------|------|
| 🔴 红色圆点 | 局部高点 (Local Highs) |
| 🔵 蓝色圆点 | 局部低点 (Local Lows) |
| 🟢 绿色实线 | 主要支撑位 |
| 🟠 橙色实线 | 主要压力位 |
| 🔴 红色实线 | 当前价格 |
| 彩色虚线 | K-Means聚类价格层级 |
| 🔴 粗红线 | POC (成交量控制点) |
| 🟢 绿色虚线 | VA (价值区域) |
| 浅绿色填充 | 价值区域范围 |

#### 左侧子图 - 成交量剖面
- 横向柱状图显示各价格区间的成交量
- 柱子越长表示该价格区间成交越活跃
- POC位置用红色线标注
- VA范围用绿色虚线标注

### 5.2 副图（下半部分）
- **RSI指标**（相对强弱指标）
  - 范围：0-100
  - >70：超买区域，可能回调
  - <30：超卖区域，可能反弹
  - 70和30位置有虚线标注

## 6. 常见问题

### Q1: 如何选择合适的参数？
**A:**
- **极值窗口**：
  - 短期交易：3-5
  - 中期分析：7-10
  - 长期趋势：12-20
- **最大聚类数**：
  - 波动小的股票：5-8
  - 波动大的股票：10-15
- **成交量分箱**：
  - 数据少（<100天）：15-20
  - 数据多（>200天）：30-50

### Q2: 置信度是什么意思？
**A:** 置信度结合了技术指标验证：
- RSI超卖/超买 +0.2
- 价格接近布林带上下轨 +0.3
- 最高1.0，最低0.5

### Q3: 机器码有效期到了怎么办？
**A:**
- 普通用户：联系管理员，管理员可以在用户管理中查看并重新启用账号
- 或者更换设备/浏览器重新登录（会注册新的机器码）

### Q4: 用户账号有效期到了怎么办？
**A:**
- 联系管理员，管理员可以在用户管理中：
  1. 删除旧账号，重新创建新账号
  2. 或者直接编辑用户的有效期（如需此功能请联系开发人员）

### Q5: 如何设置自定义日期范围？
**A:**
- 点击"开始时间"和"结束时间"输入框
- 选择你想要的日期范围
- 默认是最近一年的数据

### Q6: 图表显示乱码怎么办？
**A:** 图表已全部使用英文，不会有乱码问题。

### Q7: 为什么有些股票分析失败？
**A:** 可能的原因：
- 股票代码错误
- 网络连接问题
- 数据源暂时不可用
- 该股票数据不足（建议选择至少有60天数据的股票）
